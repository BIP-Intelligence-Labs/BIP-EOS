Implement the foundational UEOS Package Manager, including package manifests,
dependency resolution, registry services, installation workflow, cache
management, verification, CLI integration, and automated tests.

- Package manifests can be loaded and validated.
- Dependencies are resolved.
- Packages can be installed and removed.
- Package cache supports integrity verification.
- CLI commands invoke the package manager successfully.
- Unit and integration tests execute successfully.
