# Vision

BIP EOS is an AI-native Engineering Operating System designed to build,
understand, improve, and evolve intelligent software platforms.

Rather than being a single application, BIP EOS provides the engineering
foundation upon which intelligent products and domain-specific solutions
are built.

------------------------------------------------------------------------

# Mission

Build engineering systems that can:

-   Build software
-   Understand software
-   Improve software
-   Govern software
-   Evolve software
